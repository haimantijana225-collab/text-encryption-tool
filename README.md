# 🔐 Multi-Algorithm Text Encryption System

A web-based encryption tool built using Python and Streamlit.

## 📌 Features

- AES (128-bit) Encryption & Decryption
- DES Encryption & Decryption
- RSA (2048-bit) Public-Key Encryption
- Secure Random Key Generator
- RSA Key Pair Display
- Clean and Professional UI

## 🛠 Technologies Used

- Python
- Streamlit
- PyCryptodome

## 🔒 Supported Algorithms

### AES
Symmetric encryption using 16-character key.

### DES
Symmetric encryption using 8-character key.

### RSA
Asymmetric encryption using 2048-bit key pair.

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python -m streamlit run app.py
